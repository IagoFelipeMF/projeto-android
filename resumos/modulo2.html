Resumo módulo 2

Cores

Para seleção de cores o melhor é usar o adobe color para encontrar a melhor opção de paletas para o site, normalmente selecionando 3 a 5 cores.

Para fazer cores gradativas o comando utilizado é "background-image: linear-gradient" selecionando duas cores de sua preferencia.

Fontes

É possivel duas formas para integrar fontes ao seu site, sendo elas pelo download da fonte ou pela incorporação dela, normalmente feita pelo google fonts.

Tambem podemos alterar os tamanhos das fontes normais com "font-size", alem de alterar varias coisas como espaçamento e distancia entre os paragráfos com a utilização de "text-align: justify" e "text-indent" para definir a distancia inicial do paragrafo.

Seletores

Existem varios tipos de seletores entre eles:

  • id (#)

    • Identificador unico por página.

  • class (.)

    • Usado para varios identificadores.

  • div

    • Para pontuar partes em titulos(h1 ao h6).

  • span

    • Para inserir no texto (p) de forma rápida uma alteração.

E existem tambem as divisões do texto sendo:

  • Header

    • Para cabeçalho ou titulo principal.

  • Nav

    • Navegação do site.

  • Main

    • Tudo dentro do texto.

  • Article

    • Artigo especifico ou junto ao main.

  • Aside

    • Texto paralelo.

  • Footer

    • Parte inferior do site.

Pseudo classes

As pseudo classes são utilizadas para criar efeitos nas palvras com a utilização de (:) exemplo:

  • :Hover = usado para fazer algo quando mouse esta em cima da palavra.
Temos tambem os pseudo elementos (::), que servem para criar efeitos antes e depois das palavras.
  • ::After = para colocar algo depois das palavras.
Temos tambem os "childrens", que são considerados os elementos dentro dos elementos.
  • > = Para fazer a ligação de um elemento a outro.

Personalização de links

Para personalização de links podemos somente colocar diretamente no estilo para ja altera com a{} ou configurar como uma classe especifica para alterações pontuais.

Algumas utilidades são :hover, :active(para quando clicar), :visited(Para efeito diferente quando o site ja foi visitado), ::after.

Bordas

Podemos fazer bordas personalizadas com fotos, mas é pouco recomendado pelo excesso de trabalho e implementação muito especifica, mas pode ser feito colocando uma foto em tamanho 80x80 e fazendo da forma que desejar.Utilizando border-image: url('borda.png') 26 repeat, variando o repeat de acordo com a imagem escolhida.

As bordas são criadas em boxes que podem ser inline level ou block level. Sendo um bom exemplo de inline os links e os block os paragráfos. Podendo ser alterada suas formas com a utilização do comando "display".
