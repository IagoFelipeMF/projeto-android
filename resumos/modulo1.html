Adição de imagens

Comandos utilizados:

Ao colocar uma imagem no site existem 3 formas que podem ser utilizadas, sendo elas.

  1. Internas
    • Sendo baixadas previamente e colocadas como anexo direto na pasta.
  2. De uma pasta interna
    • Anexando a imagem baixada em uma pasta criada junto ao site.
  3. Externa
    • Pegando o endereço e adicionando ao site.

Para todas essas formas é necessario a utilização do comando img.

Sobre listas

Ordenadas

São as listas numéricas e todos os itens dentro sendo colocados dentro das <li> serão organizados podendo ser alterados com type 1,A,a,I,i. O número inicial tbm pode ser alterado utilizando o comando start e definindo o número de inicio.

Não ordenadas

São as listas com pontos, normalmente utilizadas para produtos, todos os itens dentro sendo colocados dentro das <li> serão organizados podendo ser alterados com type disc, circle ou square.

Sobre links

Internos, Externos e Downloads

Internos

São os links que fazemos para acessar outras páginas dentro do nosso site. Sendo necessaria a criação de uma outra página e inserindo saeu link dentro da tag <a> no href.
Dentro do comando é possível colocar o comando de target(que é utilizado para definir como o site vai se comportar ao clicar no link sendo _self(para abrir diretamente nesse navegador), _blank(para ser aberto em outra aba) sendo esses os mais utilizados) e o comando de rel(que é para definir como o site deve ver o link sendo next(para proximas paginas), prev, unfollow(quando não há plena confiança no site ao qual vc destina o usuario), external(para sites externos)).

Externos

São os links utilizados para acessar outros sites ainda sendo utilizado a tag <a> com a adição do link do site no href, e utilizando do comando _blank dentro do target.

Downloads

Para se fazer links de download, é necessario que voce tenha baixado o arquivo/foto/video do que deseja deixar para download e colocar a tag <a> com o comando de download adicionando aqui o nome do seu documento, e colocando o comando type com application/pdf para pdf's para outras formas como .zip são utilizados seus respectivos applications então sempre conferir no site iana.org

Adicionando imagens dinâmicas,áudios e videos

Imagens dinâmicas

É utilizado para criar diferentes imagens ou tamanhos da mesma imagem dependendo do tamanho da tela do usuario.
As mais utilizadas são 1000px(monitores ou tvs), 700px(tablets) e 400px(celulares) normalmente. A tag utilizada para isso é a <picture>, sendo adicionada nela a tag source media type. 

            
    picture>
    source media="(max-width: 750px)" srcset="imagens/fotop.png" type="image/png">
            source media="(max-width: 1050px)" srcset="imagens/fotom.png" type="image/png">
    img src="imagens/fotog.png" alt="imagem flexivel">

        /picture>

Áudios

Para utilizar e adicionar audio no site é necessario ter baixado o som para que ele possa ser baixado ou reproduzido pelo usuario.
Usando a tag <audio> podendo ser feito de duas formas, se o som for mp3 normalmente ele funcionara em qualquer site e só precisara ser adicionado dentro do src da tag audio, mas se quiser tambem pode ser feito como o exemplo: 


                audio preload="metadata" autoplay controls loop>
                source src="midia/som.mp3" type="audio/mpeg">
                source src="midia/som.ogg" type="audio/ogg">
                source src="midia/som.wav" type="audio/wav">
                
Infelizmente seu navegador não consegue reproduzir audio clique aqui para baixar o mp3.

                        /audio>

Videos

Localmente

São videos baixados e adicionados diretamente no seu site utilizando a tag <video>, tendo auto consumo de dados pelo seu navegador. Aqui existe a necessidade de um video baixado por isso não colocarei no exemplo apenas a forma do código. 

video src="midia/meuvideomkv.mkv" width="500" autoplay loop controls>/video >

Remotamente

São videos que estão no seu site mas que não estão baixados diretamente nele, não tendo consumo de dados local. Eles são pegos diretamente no site ao clicar em compartilhar e depois em incorporar sendo adicionado diretamente no código do seu site.

Estilos Inline, Internos e Externos

Inline

São os comandos de estilo <style> sendo adicionado diretamente na linha ao qual vai fazer alteração.
Não é recomendado para sites profissionais.

Internos

É utilizado o comando <style> no começo do código condicionando por meio de colcheias todos os componentes em que vai se fazaer a alteração.
É recomendado para sites de páginas únicas.

Externos

Aqui o comando <style> é criado em uma "página" extra como style.css e nele serão feitas todas as alterações que serão colocadas sempre no começo do código influenciando todas as paginas em que estiver inserido.
É o mais recomendado para todos os estilos de site sem excessão, pela sua rapida forma de alteração e manutenção.

Na pagina style.css não é necessaria utilização do código <style> pois ja está em uso sendo necessario apenas a alocação dos elementos que serão modificados Ex: head{ algo aqui}, h1{ algo aqui},p{ algo aqui} e assim sucessivamente, podendo alterar coisas como cor, cor de fundo, alinhamento de texto, fonte, entre outros.
